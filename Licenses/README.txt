[ 한국어 ]

이 폴더는 dodoRynx 및 함께 포함된 서드파티 구성요소의
라이선스 및 고지 문서를 포함합니다.

구성:

Licenses/
├─ LICENSE.txt
├─ DISCLAIMER.txt
├─ README.txt
├─ THIRD_PARTY_NOTICES.txt
└─ third_party/

third_party 폴더에는 배포본에 포함된 외부 라이브러리의
라이선스 원문이 포함됩니다.

참고:
- 본 프로그램은 PyInstaller onedir 방식으로 배포됩니다.
- Python 런타임 및 PySide6/Qt 구성요소가 배포본에 포함됩니다.
- 프로그램 정보 탭에서 "라이선스 폴더 열기" 버튼을 통해
  이 폴더를 열 수 있습니다.

재배포 규칙:
소스 또는 바이너리를 배포할 경우
반드시 Licenses 폴더 전체를 포함해야 합니다.

[ English ]

This directory contains license and legal information
for dodoRynx and bundled third-party components.

Structure:

Licenses/
├─ LICENSE.txt
├─ DISCLAIMER.txt
├─ README.txt
├─ THIRD_PARTY_NOTICES.txt
└─ third_party/

The "third_party" folder contains license texts
for bundled external libraries.

Notes:
- This program is packaged using PyInstaller (onedir).
- Python runtime and PySide6/Qt components are included.
- The application provides an "Open License Folder" button
  in the About tab.

Redistribution rule:
When redistributing this software (source or binary),
the entire "Licenses" folder must be included unchanged.
